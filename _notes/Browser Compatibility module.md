# 이 모듈은 마이너 모듈입니다.

이 마이너 모듈의 목적은 모든 추가적인 웹 브라우저의 호환성을 향상시키는데 있습니다. 주요 기능과 목적은 다음을 포함합니다:

- 추가 웹 브라우저를 지원하게끔 브라우저 확장 기능을 제공해서 사용자가 응용 프로그램에 원활하게 액세스하고 사용할 수 있습니다.

- 웹 애플리케이션이 올바르게 작동하고 새로 지원되는 브라우저에서 올바르게 표시되도록 철저한 테스트와 최적화를 수행하십시오

- 추가된 웹 브라우저에서 발생할 수 있는 어떠한 호환성 문제나 불일치라도 해결하세요.

### 이 모듈은 웹 어플리케이션에 추가적인 브라우저 지원을 통해서 유저에게 많은 브라우저 선택을 주는 것에 목적을 주고 있습니다.
